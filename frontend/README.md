# Frontend React

Interface inicial do marketplace com descoberta por swipe, status de escrow, reputacao verificavel e conexao WalletConnect.

## Rodar

```bash
npm install
npm run dev
```

## Principais telas/componentes

- `SwipeDeck`: descoberta de jobs/freelancers.
- `EscrowTimeline`: milestones, timeout e disputa.
- `ReputationPanel`: metricas agregadas e tierizadas.
- `WalletBar`: conexao Web3 via WalletConnect.
